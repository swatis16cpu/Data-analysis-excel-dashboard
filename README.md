# 📊 Sales Dashboard

An interactive **Sales Dashboard** built using **Microsoft Excel** to analyze sales performance across different years, categories, states, and customers. The dashboard provides dynamic visualizations that help users identify trends, monitor profits, and make data-driven business decisions.

---

## 🚀 Features

- 📅 Year-wise filtering (2014–2017)
- 🛍️ Category slicer (Furniture, Office Supplies, Technology)
- 📈 Profit trend over time
- 🗺️ State-wise sales analysis
- 👥 Customer count visualization
- 🏆 Top 5 customers by profit
- 📊 Monthly sales trend
- 📦 Sales by sub-category

---

## 📈 Dashboard Insights

- Phones generated the highest sales among all sub-categories.
- Technology products contributed the highest overall profit.
- Sales peaked during November and December.
- California recorded the highest sales.
- A small group of customers contributed a significant share of total profit.
